#像素屋扫码APP
##隐私条款
1. 信息收集

我们仅在以下必要情况下收集信息：
• 为实现二维码扫描功能，应用需要获取您设备的相机使用权限
• 为准确展示对应卡牌内容，应用会记录您扫描过的二维码标识（该标识仅用于匹配卡牌内容，不包含任何个人身份信息）
• 我们不会收集您的地理位置、通讯录、通话记录等与服务无关的个人信息

2. 信息使用

• 扫描记录仅用于提供卡牌对应内容的展示服务，确保您再次查看时能快速访问
• 我们可能对扫码数据进行匿名化统计分析，用于优化产品体验和服务质量，此类统计数据不会涉及任何可识别个人身份的信息
• 未经您的明确许可，我们不会将您的任何信息用于本条款未载明的其他用途

3. 信息存储与保护

• 您的扫描记录仅存储在本地设备或加密的服务器中，我们采取合理的技术措施保障信息安全
• 我们会定期清理过时的扫描记录，减少信息存储风险
• 所有接触用户信息的员工均需遵守严格的保密义务

4. 信息共享与披露

我们不会向任何第三方出售、出租您的信息

仅在以下情况下可能披露信息：
• 获得您的明确书面许可
• 依据法律法规、司法机关或行政主管部门的强制性要求
• 为保护我们的合法权益、用户安全或社会公共利益所必需

5. 第三方服务

• 卡牌二维码指向的网页内容可能包含第三方服务链接（如图片存储、视频播放等）
• 这些第三方服务有其独立的隐私政策，我们不对第三方的信息处理行为负责，建议您查看相关第三方的隐私政策

6. 未成年人保护

• 若您是未满 18 周岁的未成年人，应在监护人指导下使用本应用
• 我们特别注重保护未成年人隐私，不会额外收集未成年人的个人信息
• 未成年人的个人信息监护人有权要求我们进行更正或删除

7. 您的权利

• 您可以随时在设备设置中关闭相机权限，停止扫码功能的使用
• 您可以在应用内清除所有扫描记录
• 如对您的信息处理有疑问，可联系我们进行咨询

8. 条款更新

我们可能会不时更新本隐私条款，更新后的条款将在应用内显著位置公示。继续使用本应用即视为您接受更新后的条款。
